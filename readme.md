# cosmicmocha
A lightweight portfolio website made for YSDN1006. [Live Demo](http://design.sheridanc.on.ca/~bd18w301/).

## Features

* Smooth Scrolling
* CSS Animated Item Containers
* Full Viewport Navigation Pane
